#Tooling github project
You are designing a suite of elite Linux Python3 command-line tools intended for autonomous coding agents and AI software engineering systems.

GOAL:
Reduce LLM token consumption while increasing coding/debugging velocity.

Requirements:
- Each tool must be a standalone Python3 Linux CLI utility
- Output must be concise JSON by default
- Minimize verbosity
- Optimize for large repositories
- Handle real-world enterprise codebases
- Focus on:
  - code understanding
  - debugging
  - change management
  - context reduction
  - semantic analysis
  - autonomous repair workflows

For each tool provide:
1. Tool name
2. Purpose
3. CLI arguments
4. Example usage
5. JSON output schema
6. Internal architecture
7. Important implementation details
8. Recommended Python libraries
9. Failure cases
10. Performance optimization ideas

Then generate complete production-quality Python3 implementations.

Prioritize deterministic outputs and low token usage.
